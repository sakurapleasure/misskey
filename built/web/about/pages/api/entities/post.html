Post

投稿を表します。

Properties

NameTypeDescription
appApp投稿したアプリ
app_idID投稿したアプリのID
created_atDate投稿日時
idID投稿ID
is_likedBooleanいいね したかどうか
likes_countNumberいいね数
media_idsID[]添付されたメディアのIDの配列
mediaDriveFile[]添付されたメディアの配列
replies_countNumber返信数
reply_toPost返信先の投稿
reply_to_idID返信先の投稿のID
repostPostRepostした投稿
repost_countNumberRepostされた数
repost_idIDRepostした投稿のID
textString本文
userUser投稿者
user_idID投稿者のID
